Para evitarlo, registra en server/index.ts una ruta explícita ANTES de montar el middleware de Vite (dev) y antes del fallback estático de la SPA (producción):

ts
import path from "path";

app.get(["/cafeoriente", "/cafeoriente/"], (_req, res) => {
  const file =
    process.env.NODE_ENV === "production"

});
